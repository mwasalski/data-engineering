# gitignore
List of files or directories to be ignored while pushing to repo. 

## Example of a .gitignore file:

```md
# Ignore node_modules directory
node_modules/

# Ignore log files
*.log

# Ignore compiled files
*.class

# Ignore environment variables
.env
```
